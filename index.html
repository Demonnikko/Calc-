Профиль
Ставка наценки при оплате по безналу (чтобы компенсировать налоги/эквайринг).
Используется для расчёта «Чек на руки» и «Чек для клиента».
Скидка за объём
Автоматически применяется когда клиент заказывает несколько услуг. Например: 10% скидка при заказе от 2 услуг.
Сгорающий бонус
Старая цена — зачёркивается в КП. Новая цена — если 0, то "БЕСПЛАТНО", если больше 0, то показывается эта цена.
Например: Старая 15,000 → Новая 10,000 = скидка 5,000. Или: Старая 15,000 → Новая 0 = БЕСПЛАТНО.
